Accelerometer

Specification: http://www.phidgets.com/products.php?category=5&product_id=1059 Source File: accelerometer.cpp Example: accelerometer_client.cpp Published Topics Default topic: phidgets/accelerometer <accelerometer_params> This can also be altered by specifying topic_path, name and serial_number as parameters. Launch Parameters topic_path (str, default: "phidgets/") This sets the topic path for the device name (str, default: "accelerometer") An optional name for the device. The complete topic is given as the topic path plus the name. serial_number (int, default: -1) The serial number for the device. A value of -1 indicates that any serial number is acceptable. Use this with caution as there is no guarantee of which device gets connected if there is more than one device connected. sensitivity (float, default: 0.0) This is the amount that an acceleration value must change to trigger an event and publish new data to the topic. The default of 0.0 generates events and publishes to the topic as fast as the hardware can trigger events (our tests show ~62HZ) x_axis_id (int, default: 0) The accelerometer hardware provides values for axes that are indexed 0-2. Depending on how the device is mounted, the x,y,z mapping can change. Use this parameter to specify the x-axis. y_axis_id (int, default: 1) The accelerometer hardware provides values for axes that are indexed 0-2. Depending on how the device is mounted, the x,y,z mapping can change. Use this parameter to specify the y-axis. z_axis_id (int, default: 2) The accelerometer hardware provides values for axes that are indexed 0-2. Depending on how the device is mounted, the x,y,z mapping can change. Use this parameter to specify the z-axis.

accelerometer_params

Header header Message header float64[] acceleration Acceleration value for each axis of motion. float64[] orientation Tilt and toll angles in degrees. time[] stamp Timestamp for the last measurement taken for each axis of motion.

encoder_params

Header header Message header. int32 index Index number of the encoder. int32 count Returned count from the encoder. int32 count_change Change in encoder count between this and the previous value. int32 time Time when the encoder count was taken.

interface_kit_params

Header header int32 index int32 value_type int32 value

ir_params

Header header int32[] code int32[] data byte bitcount int32[] head int32[] zero int32[] one int32 trail int32 gap int32[] repeat int32 duty_cycle int32 carrier_frequency

joystick_params

Header header Message header. float32[] axes Positions of the horizontal and vertical joystick axes in the range -1 to 1. int32[] buttons States of joystick buttons (0 or 1).

led_params

Header header Message header. byte[] brightness Brightness value for each LED, in the range 0-100.

motor_params

Header header Message header. int32 index Index number of the motor (0 or 1). int32 value_type Type of value. 1 = Motor inputs 2 = Velocity 3 = Current float32 value Value returned.

phsensor_params

Header header Message header. float32 value Returned PH value. float32 potential Returned potential value.

rfid_params

Header header Message header. bool gained Whether the RFID has been gained (read) or lost. string tag The returned RFID tag.

servo_params

Header header Message header. int32 index Index number of the servo. bool engage Whether the servo is engaged/energised. float32 position Returned position of the servo. float32 speed Returned speed of the servo. float32 acceleration Returned acceleration of the servo.

spatial_params

Header header Message header float64[] acceleration Returned linear acceleration values for each axis of motion. float64[] orientation Tilt and roll angle in degrees. float64[] angular Returned angular position values for each axis of motion in degrees. float64[] magnetic Returned magnetic (compass) parameters. float64[] timestamp Timestamp value for each measurement in seconds.

stepper_params

Header header Message header. int32 index Index number of the motor. int64 position Position in steps.

textlcd_params

Header header Message header. string[] row Two rows of text to be displayed. byte contrast Sets the contrast. bool cursor Enables the cursor. bool blink Enables cursor blinking.
